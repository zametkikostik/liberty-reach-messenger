## 🎯 КАК ИСПОЛЬЗОВАТЬ

### 1. Создать приватный чат

```
1. Нажми "+" (FAB)
2. "Приватный чат"
3. Выбери контакт
4. ✅ Чат создан
5. Пиши сообщения (E2EE)
```

### 2. Создать группу

```
1. Нажми "+"
2. "Группа"
3. Введи название
4. Выбери участников
5. ✅ Группа создана
```

### 3. Создать канал

```
1. Нажми "+"
2. "Канал"
3. Введи название
4. ✅ Канал создан
```

### 4. Отправить сообщение

```
1. Открой чат
2. Введи текст
3. Нажми Send
4. 🔐 E2EE шифрование
5. ✅ Сообщение отправлено
```

### 5. Закрепить сообщение

```
1. Long press на сообщении
2. "Закрепить"
3. 📌 Появится индикатор
```

### 6. Сохранить в избранное

```
1. Long press
2. "Сохранить в избранное"
3. ⭐ Кнопка в списке чатов
4. ✅ Все сохранённые
```

### 7. Таймер самоуничтожения

```
1. Long press
2. "Таймер"
3. Выбери время
4. ⏱️ Автоудаление
```

## 🔐 БЕЗОПАСНОСТЬ

### E2EE Шифрование

```dart
// Отправка
final encrypted = _e2eeService.encryptMessage(text);

// Получение
final decrypted = _e2eeService.decryptMessage(encryptedText);
```

### Ключи

- AES-256-GCM для шифрования
- X25519 для обмена ключами
- Zero-Knowledge архитектура

## 📡 ПОТОКИ (STREAMS)

### Обновления чатов

```dart
_chatService.chatsStream.listen((chats) {
  setState(() => _chats = chats);
});
```

### Обновления сообщений

```dart
_chatService.messagesStream.listen((messages) {
  setState(() => _messages = messages);
});
```
