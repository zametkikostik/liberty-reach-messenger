## Общая информация

**Liberty Reach** — децентрализованный P2P мессенджер с открытым исходным кодом, разработанный в соответствии с Регламентом ЕС 2016/679 (General Data Protection Regulation).

### Правовое основание

- **Регламент (ЕС) 2016/679** (GDPR) — статьи 5, 6, 25
- **Закон за защита на личните данни** (Република България) — чл. 1, 3
- **Директива ePrivacy 2002/58/EC** — конфиденциальность электронных коммуникаций

### Контроллер данных

Поскольку Liberty Reach использует **децентрализованную архитектуру без центральных серверов**, проект **не является контроллером или обработчиком персональных данных** в соответствии со ст. 4 GDPR.

**Все данные хранятся исключительно на устройствах пользователей.**

## Zero-Knowledge Архитектура

### Принцип минимизации данных (ст. 5(1)(c) GDPR)

| Категория данных | Собирается? | Хранение |
|-----------------|-------------|----------|
| Телефонные номера | ❌ Нет | — |
| Email-адреса | ❌ Нет | — |
| Геолокация | ❌ Нет (без явного согласия) | Только на устройстве |
| Метаданные сообщений | ❌ Нет | — |
| История переписки | ❌ Нет | Только на устройстве |
| IP-адреса | ❌ Нет | P2P маршрутизация |
| Контакты | ❌ Нет | Только на устройстве |

### Технические принципы

1. **End-to-End Encryption (E2EE)** — AES-256-GCM + X25519
2. **Perfect Forward Secrecy** — одноразовые ключи
3. **Memory Zeroization** — немедленная очистка памяти
4. **No Central Servers** — децентрализованная сеть libp2p
5. **Local Storage Only** — все данные на устройстве пользователя

## Ваши права

Как пользователь, вы имеете следующие права по GDPR:

### Право на доступ (ст. 15)

Вы можете запросить копию всех данных, хранящихся на вашем устройстве:

```bash
# Экспорт данных
curl http://localhost:3000/api/identity/self
```

### Право на исправление (ст. 16)

Вы можете изменить свои данные в любой момент через настройки приложения.

### Право на удаление / "Право на забвение" (ст. 17)

**Мгновенное удаление всех данных:**

```bash
# GDPR Zeroize endpoint
curl -X POST http://localhost:3001/api/v1/admin/zeroize \
  -H "X-Peer-ID: ваш_peer_id"
```

**Что удаляется:**
- Все сообщения и чаты
- Ключи шифрования
- Метаданные геолокации
- История звонков
- Кэш IPFS/Pinata
- Доверенные контакты

### Право на переносимость данных (ст. 20)

**Экспорт идентичности:**

```bash
# Файл identity.key можно перенести на новое устройство
cp ~/.liberty_reach/identity.key /new-device/
```

### Право на ограничение обработки (ст. 18)

Вы можете отключить любые функции в настройках:
- Геолокация (Family Safety)
- AI-интеграции
- Web3-кошелёк
- P2P-обнаружение (mDNS)

### Право на возражение (ст. 21)

Вы можете полностью удалить приложение и все данные:

```bash
# Удаление приложения и данных
rm -rf ~/.liberty_reach/
rm -rf ~/liberty-reach-messenger/
```

## Технические меры защиты

### Криптографические примитивы

| Алгоритм | Назначение | Уровень безопасности |
|----------|------------|---------------------|
| AES-256-GCM | Шифрование сообщений | 256 бит |
| X25519 | Обмен ключами | 128 бит |
| Ed25519 | Цифровые подписи | 128 бит |
| HKDF-SHA256 | Деривация ключей | 256 бит |
| HMAC-SHA256 | Аутентификация API | 256 бит |

### Защита памяти

```rust
// Немедленная очистка секретных ключей
use zeroize::Zeroize;

let mut secret_key = generate_key();
// Использование ключа...
secret_key.zeroize(); // Уничтожение в памяти
```

### Hardware Binding

- **identity.key** — привязка к устройству
- **Права 600** — только владелец читает (Unix)
- **Secure Enclave** — поддержка TPM/Secure Element

### Anti-Brute Force

- Rate Limiting запросов
- Exponential Backoff
- Account Lockout

## Family Safety & Geo-Discovery

### Правовое основание: ст. 6(1)(a) — Согласие

**Функция Family Safety требует явного двустороннего согласия:**

1. Пользователь подтверждает доверенный контакт
2. Пользователь активирует обмен геопозицией
3. Пользователь может отключить в любой момент

### Технические гарантии

- **Только текущая позиция** — история не сохраняется
- **E2EE шифрование** — доступ только у доверенных контактов
- **SOS-сигнал** — экстренная рассылка местоположения
- **Отключение в 1 клик** — мгновенная деактивация
