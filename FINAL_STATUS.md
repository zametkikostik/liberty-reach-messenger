## ✅ ПРОВЕРКА РАБОТОСПОСОБНОСТИ

### 1. Сборка

```bash
./build.sh
# Должно завершиться успешно
```

### 2. Запуск Desktop

```bash
./build/liberty_reach_desktop
```

**Проверить:**
- [ ] Окно открылось
- [ ] Контакты видны (Алиса, Борис, Чарли)
- [ ] Статус "Онлайн ✓"
- [ ] Можно выбрать контакт
- [ ] Можно написать сообщение
- [ ] Сообщение отображается в чате
- [ ] Кнопки вызовов активны

### 3. Запуск CLI

```bash
./build/liberty_reach_cli
```

**Проверить команды:**
```
/help          # Показывает помощь
/profile       # Показывает профиль
/mesh          # Показывает Mesh статус
/encrypt тест  # Тестирует шифрование
```

### 4. Тесты

```bash
cd build
./crypto_tests   # Должен показать "passed"
./voip_tests     # Должен показать "passed"
./mesh_tests     # Должен показать "passed"
```
