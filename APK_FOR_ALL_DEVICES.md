## 🆘 КАК ОПРЕДЕЛИТЬ АРХИТЕКТУРУ

### Через приложение

1. Установи **CPU-Z** или **AIDA64**
2. Открой → раздел "System" или "CPU"
3. Найди "Instruction sets" или "Architecture"
4. Если видишь **arm64-v8a** → твой APK #2
5. Если видишь **armeabi-v7a** → твой APK #3

### Через настройки

1. Настройки → О телефоне
2. Модель процессора
3. Google: "[модель процессора] architecture"
4. arm64 → APK #2
5. arm v7 → APK #3
