## 🎯 КАК ЭТО РАБОТАЕТ

### 1. Обнаружение устройств (mDNS)

```
Устройство 1        Устройство 2
     │                   │
     ├─ mDNS Broadcast ──┤
     │                   │
     └─ "Я здесь!" ──────┘
```

**В локальной сети:**
- Устройства автоматически находят друг друга
- Не нужен центральный сервер
- Работает без интернета

### 2. Прямое соединение (WebRTC)

```
Устройство 1        Устройство 2
     │                   │
     ├─ WebRTC Handshake ┤
     │                   │
     └─ Direct P2P ──────┘
```

**После обнаружения:**
- Прямое соединение между устройствами
- Шифрование Noise protocol
- Мультиплексирование Yamux

### 3. Отправка сообщений (Gossipsub)

```
Отправитель          Получатель
     │                   │
     ├─ Gossipsub Msg ───┤
     │                   │
     └─ E2EE Encrypted ──┘
```

**Сообщения:**
- Публикация в topic
- Подписчики получают
- E2EE шифрование

## 📱 ИСПОЛЬЗОВАНИЕ

### 1. Запуск P2P

При запуске приложения:
```dart
// main.dart
final p2pService = P2PNetworkService.instance;
await p2pService.start(userId: 'user_123');
```

**Автоматически:**
- Генерация Peer ID
- Запуск mDNS
- Ожидание пиров

### 2. Обнаружение пиров

Откройте:
```
Чаты → 📡 P2P Peers (иконка вверху)
```

**Отобразится:**
- Список устройств в сети
- Статус (онлайн/офлайн)
- Кнопка для чата

### 3. Начало чата

1. Нажмите на пире
2. Откроется ChatScreen
3. Пишите сообщение
4. 🔐 E2EE шифрование
5. ✅ Отправлено через P2P

## 🔐 БЕЗОПАСНОСТЬ

### E2EE Шифрование

```dart
// Отправка
final encrypted = _e2eeService.encryptMessage(text);
await p2pService.sendMessage(
  targetPeerId: peerId,
  encryptedText: encrypted,
);

// Получение
final decrypted = _e2eeService.decryptMessage(encryptedText);
```

### Ключи

- AES-256-GCM для сообщений
- X25519 для обмена ключами
- Noise protocol для транспорта
