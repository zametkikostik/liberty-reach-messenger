## 🚀 БЫСТРЫЙ СТАРТ

### 1. Сборка

```bash
cd /home/kostik/liberty-reach-messenger
./build.sh
```

### 2. Запуск Desktop

```bash
./build/liberty_reach_desktop
```

**Интерфейс включает:**
- ✅ Чаты и контакты
- ✅ Каналы и боты
- ✅ Кошелек (кнопка 💰)
- ✅ Стикеры
- ✅ Настройки

### 3. Использование кошелька

```
1. Открыть вкладку "Кошелек"
2. Создать кошелек (запомнить recovery phrase!)
3. Получить адрес для нужного блокчейна
4. Отправить/получить криптовалюту
5. Сделать Swap или Staking
```

### 4. Создание канала

```
1. Меню → Создать канал
2. Ввести название и описание
3. Выбрать тип (публичный/приватный)
4. Пригласить подписчиков
5. Публиковать посты
```
