## 🎯 КАК ИСПОЛЬЗОВАТЬ

### 1. Обычный чат

```
1. Открой приложение
2. Введи имя пользователя
3. Нажми "+" → "Новый чат"
4. Выбери контакт
5. Пиши сообщения ✅
```

### 2. Групповой чат

```
1. Нажми "+" → "Группа"
2. Введи название
3. Выбери участников
4. ✅ Группа создана
```

### 3. Канал

```
1. Нажми "+" → "Канал"
2. Введи название
3. ✅ Канал создан
4. Подписчики получают сообщения
```

### 4. 7 TAP (Админка)

```
1. Настройки
2. 7 тапов по версии приложения
3. Введи: YourSecureRandomPassword2026!
4. ✅ Админка открыта
```

### 5. Закрепить сообщение

```
1. Long press на сообщении
2. "Закрепить"
3. ✅ Появится 📌 PINNED
```

### 6. Избранное

```
1. Long press на сообщении
2. "Сохранить в избранное"
3. ⭐ Кнопка в списке чатов
4. ✅ Все сохранённые сообщения
```

### 7. Отложенное сообщение

```
1. Напиши сообщение
2. Нажми 📅 (календарь)
3. Выбери дату и время
4. ✅ Запланировано
```

## 🆘 ПРОБЛЕМЫ

### "System Configuration Error"

**Причина:** ADMIN_MASTER_KEY не внедрён в APK  
**Решение:** Установи новый APK (собранный с ключами)

### "Пакет недействителен"

**Причина:** Старый APK без подписи  
**Решение:** Установи новый подписанный APK

### Чат не работает

**Причина:** P2P сеть не запущена  
**Решение:** Rust-ядро требует дополнительной настройки
